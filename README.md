# AbacusAurora

This repository holds the parameter files and scripts that define the cosmologies, simulation inputs, and job orchestration for Abacus on Aurora.

## Provenance and Tracking
The hash of this repo is recorded by the job scripts for provenance. To ensure that the hash actually corresponds to the job content, the scripts check that no changed or untracked files are present. This can be a nuisance, of course, if one is iterating for development. To facilitate that workflow, there is an `untracked/` directory where one can put "dirty" inputs one doesn't want to commit or track.
